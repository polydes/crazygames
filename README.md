# Crazygames SDK Extension

[Website/Releases](http://www.polydes.com/repo/engine/com.crazygames) |
[Issues](https://github.com/polydes/crazygames/issues)

</div>

This [Stencyl](https://www.stencyl.com/) extension is a wrapper around the Crazygames [HTML5 SDK](https://developer.crazygames.com/sdk#HTML5) for games intended for upload to the [crazygames.com](https://www.crazygames.com/) game portal.

## Using this Stencyl extension

This is a free extension. Installation instructions and guides can be found [here](http://www.polydes.com/repo/engine/com.crazygames/).
