This extension provides integration with the Crazygames [HTML5 SDK](https://developer.crazygames.com/sdk#HTML5) for games intended for upload to the [crazygames.com](https://www.crazygames.com/) game portal.

Most features of the HTML5 SDK are supported, including:
- ✔ ad breaks (midgame and rewarded)
- ✔ invite links
- ✔ happy time
- ✔ adblock detection
- ✔ gameplay events
